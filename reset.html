Стирание данных

На компьютере Mac с чипом Apple или на компьютере Mac с процессором Intel и чипом безопасности Apple T2 Security Chip можно использовать Ассистент стирания для восстановления заводских настроек Mac перед обменом или продажей устройства. Ассистент стирания можно использовать для стирания Mac перед переустановкой macOS. Ассистент стирания удаляет весь контент, настройки и установленные приложения.

ПРЕДУПРЕЖДЕНИЕ. При стирании компьютера Mac вся информация с него удаляется. Перед началом стирания выполните резервное копирование содержимого компьютера Mac с помощью Time Machine.

  1. Нажмите на значок Apple > «Системные настройки».
  2. В строке меню выберите «Системные настройки» > «Стереть контент и настройки».
  3. В окне «Ассистент стирания» введите данные администратора.
  4. Выберите объекты, которые будут удалены вместе с Вашим контентом и настройками.

    Если на Вашем Mac несколько учетных записей, нажмите стрелку рядом с именем Вашей учетной записи для выбора данных объектов.

  5. Нажмите «Продолжить» и следуйте инструкциям на экране.

Примечание. Если Ваша версия macOS была модифицирована, Ассистент стирания не сможет стереть данные с Вашего Mac и отобразит ошибку с указанием сначала переустановить macOS. Переустановите macOS, затем используйте Ассистент стирания.

Переустановка MacOS

ВАЖНО! Для переустановки macOS компьютер должен быть подключен к интернету.

  1. Запустите компьютер в режиме восстановления macOS.
    • На Mac с процессором Intel. Выберите меню Apple > «Перезагрузить», затем сразу же нажмите и удерживайте одно из следующих сочетаний клавиш (в зависимости от желаемого действия).
    • На Mac с процессором Intel. Выберите меню Apple > «Перезагрузить», затем сразу же нажмите и удерживайте одно из следующих сочетаний клавиш (в зависимости от желаемого действия).
      • Установка новейшей версии macOS, совместимой с Вашим компьютером. Option-Command-R.
      • Переустановка той версии macOS, которая была установлена на компьютере (включая доступные обновления). Option-Shift-Command-R.
      • Переустановка текущей версии macOS. Command-R.
  2. В окне приложения «Восстановление» выберите «Переустановить» для Вашей версии macOS, затем нажмите «Продолжить».
  3. Следуйте инструкциям на экране. На панели выбора тома выберите том, на котором установлена macOS (в большинстве случаев он будет единственным доступным диском).
